### Added
- **Core Application**: Initial release of Gemini AI Agent.
- **UI**: Modern, responsive GUI built with PyQt6 and Fluent Widgets.
- **AI Integration**: Support for Google Gemini models (Flash, Pro, Thinking).
- **Multi-Agent System (MAS)**: Orchestration for specialized sub-agents (Research, Code, File, Vision).
- **Memory**: Long-term semantic memory using ChromaDB vector store.
- **Tools**: Comprehensive toolset including file operations, system control, code analysis, and web capabilities.
- **MCP Support**: Integration with Model Context Protocol for external server connections.
- **Plugin System**: Architecture for extending functionality via plugins.
- **Conductor**: Workflow orchestration for automated tasks.
- **Documentation**: Initial README and architecture overview.

### Changed
- Updated `settings.json` structure for better configuration management.
- Refined `README.md` for better readability.
